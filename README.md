# Security Incident Mapping (SIM)

This repository contains my SEN202 Software Engineering project on **Security Incident Mapping (SIM)**. The project was developed as a web-based solution to help students, staff, and landlords report security incidents within and around the university community.

## Repository Contents
- HTML, CSS, and JavaScript source code
- Software Requirements Specification (SRS)
- Ishikawa (Fishbone) Diagram
- Use Case Diagram
